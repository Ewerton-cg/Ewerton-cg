## ▍ Key Initiatives & Deployed Systems

> *"A teoria inspira, mas é a prática de tentar, errar e otimizar que constrói ecossistemas reais."*

### 01. [Veloci BI](https://github.com/Ewerton-cg/Veloci_BI) | *Enterprise Logistical Intelligence*
Plataforma analítica desenvolvida do zero para processamento de alto volume. Centraliza a tomada de decisão logística através de pipelines de dados automatizados e interfaces executivas reativas.
* **Stack:** `Python`, `PostgreSQL`, `FastAPI`
* **Impacto:** Transformação de dados brutos operacionais em métricas visuais instantâneas.

### 02. Sentinel Crawler | *High-Throughput Data Extraction*
Motor assíncrono de extração de dados projetado com foco em resiliência. Construído testando os limites de mapeamento web para estruturar e armazenar informações em larga escala sem interrupções.
* **Stack:** `Python`, `Playwright`, `Asyncio`

### 03. Gestão-Bot v5 | *Telemetry & Anomaly Detection*
Sistema independente de monitoramento contínuo. Criado para identificar anomalias em fluxos de trabalho e alertar gestores instantaneamente diretamente na palma da mão.
* **Stack:** `FastAPI`, `Telegram SDK`
